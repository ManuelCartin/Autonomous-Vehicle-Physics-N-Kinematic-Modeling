# Autonomous Vehicle Physics & Kinematic Modeling
This repository contains the implementation of fundamental mathematical models for the control and simulation of autonomous vehicles, developed as part of my specialization at the University of Toronto.

## Description
The project focuses on modeling vehicle dynamics using Python, concentrating on how physical forces and vehicle geometry affect its motion and trajectory.

## Project Content
Bicycle Kinematic Model: Implementation of a bicycle model to approximate the vehicle's lateral motion based on steering angle and wheelbase.

Longitudinal Dynamics: Simulation of acceleration, braking, and aerodynamic drag forces to predict the vehicle's behavior in a straight line.

Trajectory Analysis: Processing of telemetry data to validate the tracking of complex trajectories (e.g., Figure Eight).

## Technologies Used
Language: Python.

Libraries: NumPy, Matplotlib (for trajectory visualization).

Environment: Jupyter Notebooks.
